# linux-homelab

A documentation-first homelab repository for tracking infrastructure, service layouts, and deployment notes.

## Structure

- `assets/` contains diagrams and screenshots.
- `compose/` contains service-specific Docker Compose stacks.
- `docs/` contains architecture, storage, networking, deployment, troubleshooting, and service notes.

## Included services

- Homepage
- Jellyfin
- Nextcloud
- Uptime Kuma
